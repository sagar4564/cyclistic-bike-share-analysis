# cyclistic-bike-share-analysis

## Project Overview

Analyze Cyclistic bike-share data to identify differences between annual members and casual riders and provide data-driven recommendations to increase annual memberships

## Tools Used

- Microsoft Excel
- Power Query
- Power Pivot
- Pivot Tables
- Data Visualization

## Process

* Collected the Cyclistic bike-share dataset
* Cleaned and transformed the data using Power Query
* Created data models using Power Pivot
* Built Pivot Tables for analysis
* Designed an interactive dashboard in Excel
* Generated business insights and recommendations

## Key Findings

- Compared riding patterns of annual members and casual riders
- Analyzed ride frequency, ride duration, and seasonal trends
- Built an interactive dashboard to present the findings clearly

## Recommendations

- Casual riders account for a significant portion of rides (36%). Offer weekend discounts or free trial memberships to encourage them to become annual members
- Since Classic Bikes are the most preferred, ensure they are available at high-demand stations, especially during peak hours
- Demand peaks around 5 PM–6 PM. Deploy additional bikes and rebalance stations before evening rush hours
- Ride volume is highest during July and August. Increase bike inventory, maintenance staff, and marketing efforts during these months
- Launch targeted membership campaigns during peak summer months (July and August), when overall ride activity is highest
